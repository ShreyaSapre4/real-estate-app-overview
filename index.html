Overview

Designed and Developed a user-friendly real estate listing website using the following technologies:

  • Backend: NodeJS, ExpressJS
  • Database: MongoDB
  • Templating: HBS (Handlebars)
  • Styling: CSS

Features

The website offers the following features for users:

  • Browse property listings easily with a user-friendly interface.
  • Explore different neighborhoods to find the perfect home.
  • Access properties for sale and rent with advanced filtering options for easy searching.
  • A simple form to list properties for sale or rent.
  • An intuitive process to input essential details and upload photos.
  • A simple authentication system to register or log user
